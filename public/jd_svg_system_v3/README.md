# Jason Donmoyer SVG Logo System V3

## Files
- `jd-icon-v3.svg` — transparent mark
- `jd-app-icon-v3.svg` — square icon for avatars and app tiles
- `logo-light-v3.svg` — full horizontal lockup for light backgrounds
- `logo-dark-v3.svg` — full horizontal lockup for dark backgrounds
- `favicon-v3.svg` — favicon-ready SVG

## What changed in V3
- custom-drawn J and D instead of plain typed letters
- more breathing room between braces and monogram
- stronger developer/SaaS feel
- tighter overall lockup spacing
- added supporting descriptor line for website/brand usage

## Brand colors
- Navy: #0F172A
- Electric Blue: #2563EB
- Neutral Gray: #6B7280
- White: #FFFFFF

## Recommended use
- Navbar/header: `logo-light-v3.svg`
- Dark hero/footer: `logo-dark-v3.svg`
- GitHub / app icon / profile avatar: `jd-app-icon-v3.svg`
- Transparent placement on docs or slides: `jd-icon-v3.svg`
- Browser favicon: `favicon-v3.svg`

## Notes
- The icon system is fully stroke/path-based.
- The wordmark remains editable SVG text for flexibility.
- For a final immutable production version, convert text to outlines in Figma or Illustrator.
